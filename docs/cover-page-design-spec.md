# ReimagineED Branding Guide - Cover Page Design Specification

## Design Brief
Create a stunning, Madison Avenue-quality cover page that scores ≥90/100 on our rubric.

---

## Layout Structure

### Hero Image (Background)
- **File:** `assets/branding-guide/cover_hero_image_1.png`
- **Treatment:** Full-bleed background with 20% darkening overlay for text legibility
- **Gradient Overlay:** Navy (#0B1D3A) to transparent, top-to-bottom, opacity 40%

### Primary Elements (Z-index layering)

#### 1. Logo/Wordmark (Top Third, Left-aligned)
**Position:** 10% from top, 8% from left
**Content:**
```
REIMAGINED
```
**Typography:**
- Font: Space Grotesk ExtraBold (or DM Sans Black)
- Size: 96pt
- Tracking: +150 (wide letter-spacing for impact)
- Color: White (#FFFFFF)
- Effect: Subtle gold (#FFD33A) glow/shadow, 4px blur

**Styling:**
- Text treatment: Split between two lines
  - "REIMAGINE" on line 1
  - "ED" on line 2, kerned close to "REIMAGINE"
- ED gets special treatment: Gold color (#FFD33A) + tech glitch effect

#### 2. Tagline (Below Logo)
**Position:** Directly below logo, 8% from left
**Content:**
```
The Disruptor in AI Education
```
**Typography:**
- Font: Inter Medium
- Size: 24pt
- Color: Electric Blue (#00D9FF)
- Style: All caps, tracking +50

#### 3. Mission Statement (Middle-Left)
**Position:** 40% from top, 8% from left, max-width 45%
**Content:**
```
Centering Black and Latino Educators at the
Forefront of the AI Revolution in Education
```
**Typography:**
- Font: Inter Regular
- Size: 18pt
- Line Height: 1.6
- Color: White (#FFFFFF) with 90% opacity
- Style: Sentence case

#### 4. Tech Visual Elements (Right Side)
**Position:** Right third of page
**Elements:**
- Neural network node pattern (subtle, gold #FFD33A, 15% opacity)
- Digital grid lines (neon blue #00D9FF, 10% opacity)
- Floating geometric shapes (hexagons, circles - navy/gold/blue gradient)

**Style:** Abstract, tech-inspired, doesn't overwhelm hero image

#### 5. Brand Pillars (Bottom Third)
**Position:** 75% from top, spanning 8-92% width
**Content:** 5 pillars in horizontal row
```
AI LITERACY  |  WORKFORCE DEVELOPMENT  |  EQUITY  |  INNOVATION  |  COMMUNITY
```
**Typography:**
- Font: Fira Code Regular (mono font for tech credibility)
- Size: 11pt
- Color: Gold (#FFD33A)
- Style: All caps, separated by vertical bars (|)
- Spacing: Even distribution across width

#### 6. Document Title (Bottom)
**Position:** 92% from top, 8% from left
**Content:**
```
BRAND GUIDE 2025
```
**Typography:**
- Font: Space Grotesk Regular
- Size: 14pt
- Color: White (#FFFFFF) 70% opacity
- Style: All caps, tracking +100

## Technical Specifications

### Canvas
- **Dimensions:** 8.5" × 11" (letter size)
- **Resolution:** 300 DPI
- **Color Mode:** CMYK (for print), RGB (for digital)
- **Bleed:** 0.125" on all sides
- **Safe Area:** 0.5" margins

### Fonts Required
1. **Space Grotesk** (ExtraBold, Regular)
2. **Inter** (Medium, Regular)
3. **Fira Code** (Regular)

### Export Formats
- **Print:** PDF (CMYK, 300 DPI, embedded fonts)
- **Digital:** PNG (RGB, 72 DPI for web, 150 DPI for presentation)

## Design Principles

### 1. Visual Hierarchy
- Logo draws eye first (largest element, high contrast)
- Tagline and mission create narrative flow
- Pillars ground the bottom
- Tech elements add depth without distraction

### 2. Balance
- Heavy visual weight on left (text)
- Lighter visual weight on right (subtle tech elements)
- Hero image provides full-bleed energy

### 3. Contrast
- White text on darkened hero image (legibility)
- Gold/blue accents pop against navy
- Tech elements subtle (15% opacity max)

### 4. Unity
- Consistent color palette throughout
- Typography choices cohesive (geometric sans, tech mono)
- All elements serve brand archetype: The Disruptor
